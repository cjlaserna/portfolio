Vibe Check Bot

Vibe Check Bot is a comedic purpose Discord bot. This was initially created as a joke, but I've expanded it's bot commands so that it can moderate profanity and pull images from Reddit. Bot commands are on a Github repository. The bot is hosted through Repl. Uptime Robot pings the bot so that it doesn't disconnect from Repl. All of the code is open for anyone to fork. Once I finish bot development, I plan on hosting the bot through a more traditional site. I used Discord Py to create the bot. For more in-depth explanation, visit it's Github repository through my profile. It has the code open for anyone to look at and the libraries I used at the bottom of the ReadME.
